# CLAUDE.md

## Project
Simple Python calculator — used to demo the PETER framework for autonomous ADWs.

## Commands
- Run tests: `python -m pytest tests/ -v`
- Run single test: `python -m pytest tests/test_calculator.py -v`

## Rules
- All functions need docstrings
- All functions need tests
- Raise ValueError for invalid inputs (not silent failures)
- Type hints on all function signatures
